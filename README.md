# 🏡 Housing Price Prediction using Decision Tree & Random Forest Regression

A Machine Learning project that predicts housing prices using **Decision Tree Regression** and **Random Forest Regression**. This project demonstrates the complete regression workflow, from data preprocessing and exploratory data analysis (EDA) to model training, evaluation, and comparison of tree-based algorithms.

The goal is to understand how decision tree-based models can be applied to solve real-world regression problems and compare their predictive performance.

---

## 📌 Project Overview

Accurately predicting house prices is an important problem in the real estate industry. In this project, machine learning models are trained on housing data to estimate property prices based on multiple features.

The project includes:

- Data Loading
- Exploratory Data Analysis (EDA)
- Data Cleaning
- Feature Engineering
- Data Preprocessing
- Train-Test Split
- Decision Tree Regression
- Random Forest Regression
- Model Evaluation
- Performance Comparison

---

## 🎯 Objectives

- Predict house prices using regression algorithms.
- Perform exploratory data analysis to understand the dataset.
- Build Decision Tree and Random Forest regression models.
- Compare the performance of tree-based algorithms.
- Learn the complete supervised machine learning workflow for regression.

## 📂 Dataset

The project uses a housing dataset containing various property-related attributes used to predict house prices.

Typical features include:

- Property Area
- Number of Bedrooms
- Number of Bathrooms
- Stories
- Parking
- Main Road Access
- Guest Room
- Basement
- Air Conditioning
- Preferred Area
- Furnishing Status
- House Price (Target Variable)
